## Java Backend Developer

Java-разработчик с практическим опытом разработки и развёртывания backend-приложений на сервере.  
Самостоятельно поднимаю проекты от кода до работающего сервиса в продакшене.

---

### Skills

- **Java Core:** ООП, коллекции (`List`, `Set`, `Map`), исключения (checked / unchecked), Stream API, lambda-выражения, `Optional`, `equals / hashCode`, immutability
- **Java 21, Spring Boot 3:** REST API, Dependency Injection, валидация, обработка ошибок
- **Работа с сервером (Linux):**
  - деплой приложений на VPS  
  - настройка окружения и переменных среды  
  - запуск и сопровождение сервисов
- **Docker, Docker Compose:**
  - сборка Docker-образов  
  - запуск приложений с базой данных  
  - обновление сервисов без потери данных
- **Базы данных:** PostgreSQL, H2  
- **ORM:** JPA / Hibernate
- **Сборка проектов:** Maven 
- **Интеграции:** внешние API, вебхуки, платёжные системы
- **Telegram-боты:** Long Polling, роли пользователей, состояния и бизнес-логика
- **Контроль версий:** Git / GitHub (ветки, pull request)
